# Image Processing Application

A simple WPF-based image editor with basic filtering and correction capabilities.

## Features

- **Image Loading** - Supports JPG, PNG, BMP, GIF, TIF formats
- **Sobel Filter** - Detects edges and creates pencil-sketch like effects
- **Contrast Adjustment** - Enhances or reduces color intensity
- **Image Export** - Save processed images in multiple formats

## Usage

1. Click "Load Image" to open an image file
2. Apply filters:
   - **Gradient** - Applies Sobel edge detection
   - **Contrast** - Adjusts image contrast using slider
3. Save the result using "Save Image" button

## Technical Details

Built with WPF (.NET), uses pixel-level manipulation for image processing operations.
